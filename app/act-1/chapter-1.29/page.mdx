# Chapter 1.29: Relations and Matrices

<ChapterCard>
### The Briefing: Mapping the Discrete
A Relation $R$ on a set $A$ is simply a subset of the Cartesian product $A \times A$. It maps elements to other elements. A relation can be perfectly represented as an adjacency matrix $M$, where $M\_{ij} = 1$ if $(a_i, a_j) \in R$, and $0$ otherwise.

Properties of Relations:
*   **Reflexive:** Every element relates to itself. $\forall a \in A, (a,a) \in R$. (The main diagonal of $M$ is all 1s).
*   **Symmetric:** If $aRb$, then $bRa$. (The matrix $M$ is symmetric: $M = M^T$).
*   **Transitive:** If $aRb$ and $bRc$, then $aRc$. 
*   **Equivalence Relation:** A relation that is simultaneously Reflexive, Symmetric, and Transitive. It partitions the set into disjoint Equivalence Classes.
</ChapterCard>

<GateTrap>
**Trap: The Empty Relation Paradox**
Consider the empty relation $R = \emptyset$ on a non-empty set $A = \{1, 2, 3\}$.
Is it Reflexive? **No** (missing $(1,1)$).
Is it Symmetric? **Yes!** (Vacuously true. *If* $aRb$, then $bRa$. Since $aRb$ never happens, the condition is never broken).
Is it Transitive? **Yes!** (Vacuously true for the same reason).
Do not assume "empty" means "lacks all properties."
</GateTrap>

<ArchitectShortcut>
**Shortcut: Transitive Matrix Check**
To mathematically prove transitivity using the adjacency matrix $M$, you compute the boolean matrix multiplication $M^2 = M \times M$.
If wherever there is a $1$ in $M^2$, there is ALSO a $1$ in the original matrix $M$, then the relation is transitive. If $M^2$ has a $1$ somewhere that $M$ has a $0$, it is NOT transitive. 
$R$ is transitive $\iff M^2 \subseteq M$.
</ArchitectShortcut>
