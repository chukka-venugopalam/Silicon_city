# Chapter 1.9: Physical Logic

<ChapterCard>
### The Briefing: The Analog Reality of Digital
"1" and "0" do not physically exist. A logic gate is a transitor circuit that interprets continuous analog voltages. To do this reliably, we define strict voltage boundaries:
* **$V\_{IL}$ (Voltage Input Low Max):** Any voltage between 0V and this threshold is safely read as Logic 0.
* **$V\_{IH}$ (Voltage Input High Min):** Any voltage between this threshold and $V\_{CC}$ (Max Supply) is safely read as Logic 1.
* **Noise Margin:** The buffer zone between the output voltage generated by a driving gate and the required input voltage of a receiving gate ($NM_H = V\_{OH} - V\_{IH}$ and $NM_L = V\_{IL} - V\_{OL}$).

If an input voltage falls between $V\_{IL}$ and $V\_{IH}$, the behavior of the gate is undefined. This is the **forbidden zone**.
</ChapterCard>

<GateTrap>
**Trap: Idealizing the Noise Margin**
Students often assume $V\_{IL}$ of the receiver equals $V\_{OL}$ of the driver. If this were true, even 0.01V of noise on the wire would push the signal into the undefined region. A robust system requires the driver to pull the voltage lower than the receiver's threshold ($V\_{OL} < V\_{IL}$) and push it higher than the receiver's high threshold ($V\_{OH} > V\_{IH}$).
</GateTrap>

<ArchitectShortcut>
**Shortcut: Reading Logic Families**
When comparing logic families (TTL vs CMOS), memorizing exact numbers isn't always necessary for GATE, but you must know the trends. 
CMOS has a much wider logic swing (rail-to-rail) and thus superior noise margins compared to TTL. TTL $V\_{IL}$ is notoriously low (0.8V), making TTL inputs highly susceptible to ground-bounce noise.
</ArchitectShortcut>

### The Proving Ground

**Question 1 (NAT):**
A logic gate has the following specifications: $V\_{OH} = 2.4V$, $V\_{IH} = 2.0V$, $V\_{OL} = 0.4V$, and $V\_{IL} = 0.8V$. What is the High-state Noise Margin ($NM_H$) in Volts?

**Question 2 (NAT):**
If the same gate from Question 1 receives an input signal of $1.5V$, what is the defined logic state of the input? (Answer 0 for Logic 0, 1 for Logic 1, or 2 for Undefined).
